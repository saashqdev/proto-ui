### Basic Usage

``` html
<u-linear-layout>
    <u-logo></u-logo>
    <u-logo color="inverse" href="/proto-ui/" target="_blank"></u-logo>
</u-linear-layout>
```

### Add Subtitle

``` html
<u-linear-layout>
    <u-logo href="/proto-ui/">Prototype Component</u-logo>
    <u-logo color="inverse" href="https://kubevue.github.io/icon-sets" target="_blank">Icon Library</u-logo>
</u-linear-layout>
```
